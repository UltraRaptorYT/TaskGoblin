# TaskGoblin demo files

This folder contains a fictional Telegram Desktop JSON export for testing the TaskGoblin import and scan flow.

## Demo project

`safe-route-ai/result.json` represents a four-person student group building **SafeRoute AI**, a portable data and machine-learning application for analysing risky driving behaviour.

The conversation was inspired by the structure of the provided Practical AI assignment briefs. It includes:

- SQL Server storage and complex queries
- An ETL pipeline from SQL to Python
- Data cleaning and exploratory analysis
- An interactive dashboard
- Feature engineering and model optimisation
- MLflow experiment tracking
- A desktop prediction GUI
- Reports, slides, Scrum records, and peer evaluations

The messages intentionally contain confirmed assignments, decisions, changed deadlines, blockers, missing owners, and vague commitments. This gives TaskGoblin useful signals to extract and display.
